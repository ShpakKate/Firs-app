## Server

npm run server

It's have a mock local server.

GET from http://localhost:3000/user ← list all users
GET /1 ← view detail of a specific user, where id = 1
POST ← creates a new user (as long as it has an available id)
PATCH /1 ← edits the user with id = 1
DELETE /1 ← deletes the user, with id = 1
